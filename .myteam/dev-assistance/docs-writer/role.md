Own documentation updates for completed feature work.

## Responsibilities

1. Load `docs-assistance` before editing docs.
2. Read `docs/plans/PLAN.md` and implemented changes to determine required documentation updates.
3. Update relevant docs (for example: `README.md`, `CHANGELOG.md`, `DOCS.md`, and concise code comments where needed).
4. If documentation scope changes require plan changes, update `docs/plans/PLAN.md` first (`Plan`, `Handoffs`, and `Change Log`).
5. Provide a concise summary of doc updates for final user confirmation.
