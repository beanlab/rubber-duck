# Prompt Archive

This folder contains legacy or currently unreferenced prompt assets preserved for historical reference.

Runtime configuration should reference active prompts outside this folder unless explicitly migrating archived assets back into production use.

## Archived Prompt Groups

- `root/`: legacy top-level prompt files no longer wired by config/runtime.
- `stats/`: retired stats sub-agent prompt set kept for reference.
- `production-prompts/`: prompt files previously colocated with active production prompts but not referenced in current configs.
