# CS110 TA Duck

Created on: 2026-04-20
Created by: Tyler

## Details

Create a duck that can perform the function of a CS110 lecture assistant.

- What problem does the feature address?
  - Students need structured help that mirrors the CS110 lecture workflow.
- What is the intent of the feature?
  - Provide a guided, lecture-like experience with an objective and workflow.
- What details exist so far about this feature?
  - Observations from when Tyler attended the first class of the semester inform the interaction style:
    - Start high-level before diving into details.
    - Ask one focused question at a time.
    - Probe confusion explicitly.
    - Have learners articulate understanding.
    - Use analogies carefully and only when appropriate.
    - Break syntax into minimal pieces.
    - Encourage active reasoning and prediction.
    - Check gaps before explaining.
