# Simple Coding Project: To-Do List CLI App (Python)

## Project Overview

Create a simple **Command-Line To-Do List** application in Python that allows users to:
- Add tasks
- View tasks
- Mark tasks as done
- Delete tasks

All tasks should be saved to a file so the data persists between runs.

---

## Project Outline

### 1. Setup
- Create a Python file (e.g., `todo.py`)
- Use built-in libraries (`argparse`, `json`, `os`) only

### 2. Features
- [ ] `add`: Add a new task
- [ ] `list`: List all tasks (pending and done)
- [ ] `done`: Mark a task as complete
- [ ] `delete`: Remove a task
- [ ] Tasks should persist in a JSON file (`tasks.json`)

### 3. Data Structure
```json
[
  {
    "id": 1,
    "task": "Buy groceries",
    "done": false
  }
]
